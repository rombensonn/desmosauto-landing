# PHP-бэкенд формы

Файл `api/leads.php` принимает POST-заявки с сайта, валидирует имя и телефон, проверяет honeypot и сохраняет заявку в `backend/storage/leads.jsonl`.

Внешние сервисы не используются. При необходимости можно включить отправку через локальную почтовую службу сервера переменной `LEAD_MAIL_TO`.

Переменные окружения:

- `LEAD_ALLOWED_ORIGINS` — список разрешённых доменов через запятую для CORS.
- `LEAD_SALT` — соль для хеша IP.
- `LEAD_MAIL_TO` — необязательный адрес для локальной серверной почты.

На продакшене endpoint должен быть доступен как `/api/leads.php`, либо укажите путь в `NEXT_PUBLIC_LEAD_ENDPOINT`.
