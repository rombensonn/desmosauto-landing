# 12. Краткое описание проекта

ДесмосАвто - это специализированный сайт для продажи разработки сайтов автосервисам и смежным автомобильным бизнесам. Проект обещает показать демо-версию сайта за сутки без предоплаты и обязательств. Его главный смысл: сайт автосервиса должен быть не визиткой, а цифровым администратором, который объясняет услуги, показывает доверие, отвечает на частые вопросы и приводит короткие заявки.

Целевая аудитория - владельцы и управляющие СТО, шиномонтажей, детейлинг-студий, кузовных цехов, центров диагностики, техосмотра и автозапчастей. Им важны скорость запуска, понятная структура, реальные кейсы, заявки в Telegram и минимальная ручная настройка после передачи.

Визуально проект построен как премиальный светлый editorial/SaaS лендинг с авто-специализацией: почти черный текст, теплые белые поверхности, тонкие линии, оранжевый акцент `#ff5a1f`, тяжелая типографика, крупные секции, 3D PNG-модели, реальные превью проектов и спокойные, дорогие анимации. Главные анимационные системы: GSAP ScrollTrigger для reveal, horizontal pinned scroll и sticky stack; Framer Motion для FAQ, модалок и счетчиков; CSS hover для микровзаимодействий; React Three Fiber/WebGL для AEO-сцены.

Технически сайт сделан на Next.js App Router, React, TypeScript и Tailwind CSS со статическим экспортом в `out/`. Форма заявки отправляется в PHP backend `api/leads.php`, сохраняет лиды в `storage/leads.jsonl` и отправляет уведомления в Telegram. Для REG.RU Host-0 собирается готовый архив, где в корне лежат `index.html`, `.htaccess`, `_next/`, `api/leads.php`, `api/leads.config.php` и защищенная папка `storage/`.

При любой будущей работе нужно сохранять авто-контекст, короткий путь к заявке, визуальную систему, надежную Telegram-отправку и готовность к публикации на REG.RU Host-0.
